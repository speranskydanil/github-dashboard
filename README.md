# GitHub Dashboard

Simple application to view trends on github.

## <a href="http://speranskydanil.github.io/github-dashboard/demo.html">DEMO</a>

### Quick Start

1. Clone repository
2. Install Gems (bundle install)
3. Run Server (./server 3001)
4. Open http://localhost:3001/
